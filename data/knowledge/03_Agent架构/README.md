# Agent架构与原理

> 第2周（3.11 - 3.17）| 优先级：核心（Agent岗必考）

## 要搞定的知识点

- Agent的定义与核心组成（感知、规划、行动、记忆）
- ReAct模式（Reasoning + Acting，执行流程，与CoT的区别）
- Plan-and-Execute模式（先规划后执行，适用场景）
- 反思机制（Reflexion，自我纠错循环）
- 多智能体架构（主从模式、对等协作、辩论模式）
- Agent vs Workflow 的区别与选型
- Agent的评测方法
- Agent评测体系（benchmark设计、自动化测试脚本、场景回放、成功率/延迟/成本指标）
- 其他Agent框架对比（CrewAI、AutoGen、Dify各自定位、适用场景、与LangGraph的区别）
